FAQ

Common questions.

If you do not see your question here, you can extend the app or tailor the flow to your needs.

  • Is Whisper Flow free to use?

    The interface itself is just your local app code, so there is no built-in subscription here. Your actual cost depends on the Azure services and usage tied to the endpoint you configure.

  • What is Smart Typing in this project?

    Smart Typing is the optional cleanup concept shown in the landing page. It means taking a raw transcript and improving fillers, punctuation, and self-corrections before you reuse the text.

  • Does it only work on Apple Silicon or macOS?

    No. This version is a browser-based Flask app and is currently framed for a Windows-friendly workflow. The key requirement is browser microphone access plus a working Azure Whisper setup.

  • Does Whisper Flow send audio or transcripts anywhere?

    Audio is sent to the Azure endpoint you configure for transcription or translation. Outside of that, this app does not add its own remote storage or separate transcript collection layer.

  • Can I teach it names, jargon, or domain terms?

    The page already presents a custom dictionary concept, and the project can be extended to support it. That is the right place to preserve recurring names, acronyms, and domain-specific language.

  • Should I use translation or transcription mode?

    Use translation mode when you want English output from supported spoken input. Use transcription mode when you want the transcript returned in the original spoken language.

  • Can I change the record interaction later?

    Yes. Right now the app uses a browser recording button, but the UI and backend are structured in a way that can evolve into push-to-talk, shortcuts, history, export flows, or richer desktop-style behavior.
